Kata Containers
=========

An Ansible role to deploy Kata Containers on Kubernetes clusters (containerd specific)

Requirements
------------

* A Kubernetes cluster, using containerd, up and running.

Example Playbook
----------------

```sh
ansible-playbook -i /path/to/inventory/file tasks/main.yml
```
